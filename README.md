# WobbleView.Xamarin
WobbleView.Xamarin is a Xamarin.iOS port of the [WobbleView](https://github.com/inFullMobile/WobbleView) control for Swift, originally created by [Kondrat Kielar](https://github.com/kondratk).

## Requirements
- Xamarin.iOS 7.0+
- iOS 7.0+
